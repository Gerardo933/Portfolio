# 💼 Frontend Portfolio

¡Hola! Este es mi portafolio personal de desarrollo frontend, creado con **Next.js**, **TypeScript**, **Redux** y **Emotion** para estilos.
Aquí muestro mis proyectos, experiencia, habilidades y formas de contacto profesional.
###########################################################################################################################################3<w>

## 🚀 Tecnologías utilizadas

- ⚛️ **Next.js** – Framework de React para SSR y generación de sitios estáticos
- 🟦 **TypeScript** – Tipado estático para una mayor seguridad y mantenibilidad
- 🎨 **Emotion** – CSS-in-JS moderno para componentes estilizados
- 🧠 **Redux** – Manejo centralizado del estado global
- 📁 **Alias @/** – Para importar más limpio desde `src/`

---

## 📁 Estructura del proyecto

src/
├── pages/ # Enrutamiento clásico de Next.js
│ ├── index.tsx
│ ├── about.tsx
│ └── contact.tsx
├── components/ # Componentes reutilizables
├── store/ # Configuración de Redux
├── styles/ # Temas y estilos globales (con Emotion)
